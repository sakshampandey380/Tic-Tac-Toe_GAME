# 🎮 Ultimate Tic-Tac-Toe Game

A modern, feature-rich, and fully responsive version of the classic Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**.

This project combines **clean UI/UX design, animations, AI logic, and responsive development** to deliver a premium gaming experience across all devices.

---

## ✨ Features

### 🤖 Single Player Mode (AI)
- Play against a smart AI opponent  
- Strategic gameplay (not just random moves)  
- Enhances logical thinking  

---

### 👥 Dual Player Mode
- Play with friends on the same device  
- Smooth turn-based gameplay  
- Simple and fun interaction  

---

### 🔄 Mode Switching
- Toggle between **Single 🤖** and **Duo 👥** anytime  
- No page reload required  
- Instant reset when switching  

---

### 🎉 Winning Effects
- Confetti explosion on win  
- Glowing winning boxes  
- Animated winner popup (auto hides in 2 seconds)  

---

### 🏆 Scoreboard System
- Real-time score tracking  
- Separate scores for Player O and Player X  
- Keeps session-based performance  

---

### 🎬 Smooth Animations
- Game start animation  
- Button click effects  
- Hover interactions  
- Popup transitions  

---

### 🌙 Dark Mode
- Toggle between light and dark themes  
- Better viewing experience  

---

### 🔊 Sound Effects
- Click sound feedback  
- Win sound effect  
- Optional background music  

---

### 📱 Fully Responsive Design
- Works perfectly on:
  - Mobile phones 📱  
  - Tablets 📲  
  - Laptops 💻  
  - Desktops 🖥️  
- No layout breaking or overflow

### 🎨 Premium UI Design
- Gradient backgrounds

---

## 🎮 How to Play

- Select a mode:
- 🤖 Single Player (vs AI)
- 👥 Dual Player (with friend)

- Players take turns placing:
- **O** and **X**

- Win by completing:
- Row  
- Column  
- Diagonal  

- If all boxes are filled → **Draw**
